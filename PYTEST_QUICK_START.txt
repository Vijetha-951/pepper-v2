TROUBLESHOOTING:

Problem: "Python is not installed"
Solution: Install Python 3.8+ from python.org

Problem: "ModuleNotFoundError: No module named 'selenium'"
Solution: pip install -r requirements.txt

Problem: "Chrome/Firefox not found"
Solution: Install Chrome or Firefox browser

Problem: Tests timeout
Solution: Increase timeouts in config.py:
  IMPLICIT_WAIT = 30
  PAGE_LOAD_TIMEOUT = 60

Problem: Tests fail on login
Solution: Check TEST_USER credentials in config.py

COMMON PYTEST FLAGS:

-v      Verbose output
-s      Show print statements
-x      Stop on first failure
-k      Run tests matching keyword (e.g., -k "login")
-m      Run tests with marker (e.g., -m "smoke")
--tb    Traceback format (short, long, native)
