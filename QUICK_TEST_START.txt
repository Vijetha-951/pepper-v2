  ‚Üí Make sure PEPPER frontend is running on http://localhost:3000
  ‚Üí Run: npm start (in frontend directory)

Problem: "Chrome not found"
  ‚Üí Install Chrome from https://google.com/chrome
  ‚Üí Or use Firefox instead: BROWSER=firefox npm test

Problem: "Admin login fails"
  ‚Üí Verify admin user exists: admin@example.com / admin123456
  ‚Üí Check database user table

  ‚Üí Try fewer tests at once

  ‚Üí Update selectors in pepper-functional-tests.test.js
